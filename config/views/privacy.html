Privacy Policy

Last updated: 2026-04-02

This page explains how libiweb API processes request data.

Data Processed

Requests may include IP address, timestamp, endpoint path, headers, and response status for service operation and abuse prevention.

Purpose

Data is processed to provide the API, monitor availability, apply rate limits, and protect the service from unauthorized use.

Retention

Operational logs are retained only for as long as required for security and troubleshooting purposes.

Contact

For privacy-related requests, contact the API administrator through your support channel.
